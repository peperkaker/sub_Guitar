## 第二步：认识你的吉他

### 吉他各部位名称
```
        琴头 (Headstock)
          |
        弦钮 (Tuning Pegs)
          |
        琴颈 (Neck)
          |
    品丝 (Frets) ————
          |
        音孔 (Sound Hole)
          |
        琴身 (Body)
          |
        琴桥 (Bridge)
```

### 六根弦的名称（从粗到细）
- 6弦（最粗）：E（低音Mi）
- 5弦：A（La）
- 4弦：D（Re）
- 3弦：G（Sol）
- 2弦：B（Si）
- 1弦（最细）：E（高音Mi）

**记忆口诀**：Eat All Day Get Big Easy

## 第三步：正确的姿势

### 坐姿
1. 坐在椅子前1/3处
2. 背部挺直，不要驼背
3. 吉他腰部凹陷处放在右腿上
4. 琴头略高于琴身
5. 吉他与身体成45度角

### 左手姿势
1. 拇指放在琴颈后方中间位置
2. 其他四指自然弯曲
3. 用指尖按弦（不是指肚）
4. 手腕保持放松

### 右手姿势
1. 手臂自然搭在琴身上
2. 手腕放松，略微弯曲
3. 扫弦时以手腕为轴心
4. 不要过度用力

---

## 第四步：第一次调音

### 使用调音器
1. 打开调音器，夹在琴头上
2. 拨响一根弦
3. 观察调音器显示
   - 指针偏左：音低了，拧紧弦钮
   - 指针偏右：音高了，放松弦钮
   - 指针居中且显示绿色：音准了
4. 依次调好六根弦

### 标准音高
- 1弦：E（329.63 Hz）
- 2弦：B（246.94 Hz）
- 3弦：G（196.00 Hz）
- 4弦：D（146.83 Hz）
- 5弦：A（110.00 Hz）
- 6弦：E（82.41 Hz）

**提示**：新琴或新弦需要多次调音才能稳定

## 第五步：第一个练习

### 爬格子练习（手指灵活度训练）

**左手按法**：
```
1弦：1品(食指) → 2品(中指) → 3品(无名指) → 4品(小指)
2弦：1品(食指) → 2品(中指) → 3品(无名指) → 4品(小指)
3弦：1品(食指) → 2品(中指) → 3品(无名指) → 4品(小指)
...依此类推到6弦
```

**练习要点**：
- 每个音都要清晰
- 保持均匀的节奏用节拍器
- 从慢到快
- 每天练习10分钟

## 第六步：第一次和弦 - Em Am

Em和弦是最简单的和弦，只需要按两根弦！

### 和弦图示

```
Am和弦：
E|---0---
B|---1---
G|---2---
D|---2---
A|---0---
E|---x---

Em和弦：
E|---0---
B|---0---
G|---0---
D|---2---
A|---2---
E|---0---
```

### 练习步骤
1. 按好和弦
2. 从6弦到1弦，逐根拨响
3. 检查每根弦是否清晰
4. 重复练习，直到每次都能按响

---

## 第七步：第一个节奏型

### 四拍下扫
```
拍子：1   2   3   4
动作：↓   ↓   ↓   ↓
```

### 练习方法
1. 打开节拍器，设置60 BPM
2. 按住Em和Am和弦
3. 跟随节拍器，每拍向下扫一次
4. 保持均匀的力度和速度

## 本周

### 练习建议（60-90分钟）确保琴弦音准，使用节拍器

**第1周** 
- 调音
- 1～4品空按练习（70BPM）
- 1～4爬格子练习（70BPM）
- 空弦拨弦练习 （70BPM）
- 休息：5分钟
- Em和Am和弦练习 （70BPM）
- 兰花草 （70BPM）

## 学习建议

### 做到这些，进步会更快
✅ 每天坚持练习，哪怕只有30分钟  
✅ 保持正确姿势，避免养成坏习惯  
✅ 使用节拍器，培养节奏感  
✅ 录音/录像，检查自己的演奏  
✅ 不要急于求成，循序渐进  
✅ 遇到困难不要放弃，多练习  

### 避免这些错误
❌ 姿势不正确，容易受伤  
❌ 不使用节拍器，节奏不稳  
❌ 练习时间过长，手指过度疲劳  
❌ 跳过基础，直接学难曲  
❌ 不调音就练习  
❌ 三天打鱼两天晒网

## 常见问题解答

### Q1：手指按弦很疼怎么办？
**A**：这是正常现象！坚持练习1-2周，手指会长茧，就不疼了。如果太疼可以休息一下。

### Q2：和弦总是按不响？
**A**：检查三点：
1. 手指是否垂直按弦
2. 是否按在品丝旁边（不是品丝上）
3. 是否用力足够

### Q3：多久能学会一首歌？
**A**：简单的歌曲1-2周，复杂的可能需要1个月。不要着急，享受过程。

### Q4：每天应该练习多久？
**A**：建议60-90分钟。如果时间有限，30分钟也可以，但要保证每天都练。

### Q5：需要学习乐理吗？
**A**：基础乐理会有帮助，但不是必须的。可以边学边了解。
