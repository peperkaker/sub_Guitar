## 第1天：认识你的吉他

### 理论知识（30分钟）

#### 吉他的构造
```
琴头 (Headstock)
  ├─ 弦钮 (Tuning Pegs) - 用于调音
  └─ 弦枕 (Nut) - 固定琴弦

琴颈 (Neck)
  ├─ 指板 (Fingerboard)
  ├─ 品丝 (Frets) - 共19-22品
  └─ 品位标记 (Position Markers) - 通常在3、5、7、9、12品

琴身 (Body)
  ├─ 面板 (Top)
  ├─ 音孔 (Sound Hole)
  ├─ 护板 (Pickguard)
  └─ 琴桥 (Bridge) - 固定琴弦
```

#### 六根弦的标准定音
- **6弦**（最粗）：E（82.41 Hz）低音Mi
- **5弦**：A（110.00 Hz）La
- **4弦**：D（146.83 Hz）Re
- **3弦**：G（196.00 Hz）Sol
- **2弦**：B（246.94 Hz）Si
- **1弦**（最细）：E（329.63 Hz）高音Mi

**记忆口诀**：Eat All Day Get Big Easy

#### 品位说明
- 品位从琴头向琴身方向数：1品、2品、3品...
- 按弦要按在品丝旁边，不是品丝上
- 品位越高，音越高

### 实践练习（60分钟）

#### 练习1：调音（15分钟）
1. 打开调音器APP或夹式调音器
2. 从6弦开始，逐根调音
3. 拨响琴弦，观察调音器显示
4. 调到标准音高（指针居中，显示绿色）
5. 重复2-3次，确保音准稳定

**注意事项**：
- 新琴或新弦需要多次调音
- 拧弦钮时要慢，避免弦断
- 音低了往紧拧，音高了往松拧

#### 练习2：右手拨弦（20分钟）
1. 右手自然搭在琴身上
2. 用拇指拨6、5、4弦
3. 用食指拨3弦
4. 用中指拨2弦
5. 用无名指拨1弦

**练习方法**：
- 从6弦到1弦，逐根拨响
- 每根弦拨10次
- 保持均匀的力度
- 听每根弦的音色

#### 练习3：左手按弦（25分钟）
1. 左手拇指放在琴颈后方中间
2. 用食指按1弦1品
3. 右手拨响，听是否清晰
4. 依次练习2、3、4品
5. 换到其他弦重复

**要点**：
- 用指尖按弦，不是指肚
- 手指要垂直于指板
- 按在品丝旁边，不是品丝上
- 用力要适中，能发出清晰的音即可

### 今日总结

## 第2天：正确的演奏姿势

### 理论知识（30分钟）

#### 坐姿要点
1. **椅子选择**：硬质椅子，高度适中
2. **坐姿**：坐在椅子前1/3处
3. **背部**：挺直，不要驼背或靠椅背
4. **双脚**：平放地面，与肩同宽
5. **吉他位置**：
   - 琴身腰部凹陷处放在右腿上
   - 琴身与身体成45度角
   - 琴头略高于琴身（约30度）

#### 左手姿势
1. **拇指位置**：琴颈后方中间，与中指相对
2. **手掌**：不要贴在琴颈上，保持空间
3. **手腕**：自然弯曲，不要过度弯曲
4. **四指**：自然弯曲，指尖按弦
5. **指甲**：左手指甲要剪短

#### 右手姿势
1. **手臂**：自然搭在琴身上
2. **手腕**：放松，略微弯曲
3. **手掌**：悬空，不要贴在琴弦上
4. **扫弦**：以手腕为轴心，不是手臂
5. **指甲**：右手可留一点指甲（可选）

### 实践练习（60分钟）

#### 练习1：姿势矫正（20分钟）
1. 对着镜子检查坐姿
2. 保持正确姿势5分钟
3. 休息1分钟
4. 重复4次

**检查清单**：
- [ ] 背部挺直
- [ ] 吉他位置正确
- [ ] 左手拇指位置正确
- [ ] 右手自然放松

#### 练习2：空弦拨弦（20分钟）
**练习模式**：6-5-4-3-2-1-2-3-4-5-6
- 保持正确姿势
- 每根弦拨2次
- 重复10遍
- 速度：慢速，均匀

#### 练习3：左手按弦练习（20分钟）
**1品练习**：
- 食指按1弦1品，拨响
- 食指按2弦1品，拨响
- 依次到6弦
- 重复5遍

**2品练习**：
- 中指按1弦2品，拨响
- 依次到6弦
- 重复5遍

**3品、4品同理**

## 第3天：爬格子基础训练

### 理论知识（30分钟）

#### 什么是爬格子？
爬格子是吉他最基础的练习，用于：
- 训练手指独立性
- 提高手指灵活度
- 建立肌肉记忆
- 熟悉指板位置

#### 手指编号
- **1**：食指
- **2**：中指
- **3**：无名指
- **4**：小指

#### 爬格子原则
1. 一指对一品（食指1品，中指2品...）
2. 手指不要离弦太远（1-2厘米）
3. 保持手型，不要塌陷
4. 速度慢，准确性第一

### 实践练习（60分钟）

#### 练习1：横向爬格子（25分钟）
**1234模式**：
```
1弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
2弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
3弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
4弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
5弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
6弦：1品(1指) - 2品(2指) - 3品(3指) - 4品(4指)
```

**练习方法**：
- 从1弦开始，按1234顺序
- 每个音都要清晰
- 到6弦后，反向回到1弦（4321）
- 重复5遍

#### 练习2：1324模式（15分钟）
```
顺序：食指 - 无名指 - 中指 - 小指
1弦：1品 - 3品 - 2品 - 4品
（依次到6弦）
```

**目的**：训练手指独立性，特别是无名指

#### 练习3：1423模式（15分钟）
```
顺序：食指 - 小指 - 中指 - 无名指
1弦：1品 - 4品 - 2品 - 3品
（依次到6弦）
```

**目的**：训练小指力量

#### 练习4：节拍器配合（5分钟）
- 打开节拍器，设置60 BPM
- 每拍按一个音
- 跟随节拍器练习1234模式

### 今日总结
- [ ] 完成1234爬格子5遍
- [ ] 完成1324模式3遍
- [ ] 完成1423模式3遍

## 第4天：手指独立性训练

### 理论知识（30分钟）

#### 手指独立性的重要性
- 每个手指能够独立运动
- 不会互相牵制
- 提高演奏速度和准确性
- 为和弦转换打基础

#### 常见问题
1. **小指无力**：正常现象，需要加强训练
2. **无名指不独立**：与小指连动，需要专项练习
3. **手指僵硬**：练习前要热身，练习后要放松

### 实践练习（60分钟）

#### 练习1：单指力量训练（20分钟）
**食指训练**：
- 1弦1品按住，拨响10次
- 2弦1品按住，拨响10次
- 依次到6弦

**中指、无名指、小指同理**

**要点**：
- 只用一个手指按弦
- 其他手指自然弯曲，不要伸直
- 每个音都要清晰

#### 练习2：双指配合（20分钟）
**1-2指配合**：
```
1弦：1品(食指) - 2品(中指) 交替按
重复20次

**1-3指配合**：

重复20次
```

**1-4指配合**：
```
1弦：1品(食指) - 4品(小指) 交替按
重复20次
```

**2-3指、2-4指、3-4指同理**

#### 练习3：三指配合（15分钟）
**123模式**：
```
1弦：1品 - 2品 - 3品 循环
重复10次
```

**124模式**：
```
1弦：1品 - 2品 - 4品 循环
重复10次
```

**134模式、234模式同理**

#### 练习4：四指循环（5分钟）
```
1弦：1-2-3-4-3-2-1 循环
重复5次
```

## 第5天：节拍器的使用

### 理论知识（30分钟）

#### 什么是节拍器？
节拍器是帮助保持稳定节奏的工具，是练习吉他必备。

#### 节拍器的参数
1. **BPM**（Beats Per Minute）：每分钟的拍数
   - 60 BPM = 每秒1拍
   - 120 BPM = 每秒2拍
2. **拍号**：如4/4拍、3/4拍
3. **重音**：第一拍通常有重音提示

#### 常用速度
- **慢速**：60-80 BPM（初学者）
- **中速**：80-120 BPM（练习）
- **快速**：120-180 BPM（进阶）

#### 如何使用节拍器
1. 先听节拍器，感受节奏
2. 跟着节拍器拍手
3. 再配合吉他练习
4. 从慢速开始，逐渐加快

### 实践练习（60分钟）

#### 练习1：节奏感训练（15分钟）
**步骤1**：听节拍器（60 BPM）
- 闭眼听2分钟
- 感受每一拍

**步骤2**：跟着拍手
- 每拍拍一次手
- 持续3分钟

**步骤3**：跟着踏脚
- 每拍踏一次脚
- 持续3分钟

#### 练习2：空弦配合节拍器（20分钟）
**60 BPM**：
- 每拍拨一根弦
- 6-5-4-3-2-1循环
- 持续5分钟

**70 BPM**：
- 同样练习
- 持续5分钟

**80 BPM**：
- 同样练习
- 持续5分钟

#### 练习3：爬格子配合节拍器（25分钟）
**60 BPM - 每拍一个音**：
```
1弦：1品(拍1) - 2品(拍2) - 3品(拍3) - 4品(拍4)
```
- 完整爬格子5遍

**70 BPM**：
- 完整爬格子3遍

**80 BPM**：
- 完整爬格子2遍

### 今日总结
- [ ] 学会使用节拍器
- [ ] 能够跟随60-80 BPM练习
- [ ] 完成节拍器配合爬格子
