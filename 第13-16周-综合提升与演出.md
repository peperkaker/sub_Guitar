**学习目标**：

- 能够进行舞台表演

## 第13周：演唱技巧配合

### 本周目标
- [ ] 弹唱配合更加流畅
- [ ] 能够稳定演唱
- [ ] 学会《晴天》

### 每日学习计划

#### 第1天：气息控制
**理论**（30分钟）：
- 正确的呼吸方法
- 气息支撑
- 长音练习

**练习**（60分钟）：
- 腹式呼吸练习
- 长音保持
- 气息稳定性训练

#### 第2天：弹唱配合技巧
**理论**（30分钟）：
- 如何边弹边唱不跑调
- 注意力分配
- 弹唱协调训练

**练习方法**：
1. 先熟练弹奏（不唱）
2. 先熟练演唱（不弹）
3. 慢速弹唱结合
4. 逐渐提速

**练习**（60分钟）：
- 简单歌曲弹唱
- 慢速练习
- 逐步提速

#### 第3天：音准训练
**理论**（30分钟）：
- 如何保持音准
- 常见跑调原因
- 音准训练方法

**练习**（60分钟）：
- 跟唱原唱
- 录音对比
- 纠正音准

#### 第4-7天：《晴天》
**歌曲**：晴天 - 周杰伦
**难度**：★★★☆☆

**和弦进行**：
```
前奏：Am - F - C - G
主歌：C - Em - F - G
副歌：Am - F - C - G
```

**练习内容**：
- 和弦熟练
- 节奏型选择
- 弹唱配合
- 情感表达
- 完整演绎

**重点难点**：
- 副歌部分节奏较快
- 需要良好的弹唱配合
- 注意情感起伏

## 第14周：舞台表现力

### 本周目标
- [ ] 能够站立弹唱
- [ ] 提升舞台表现力
- [ ] 完成《平凡之路》

### 每日学习计划

#### 第1天：舞台表现理论
**理论**（30分钟）：
- 舞台表现的重要性
- 肢体语言
- 眼神交流
- 情感传达

**要点**：
- 自信的姿态
- 适当的表情
- 与观众的互动
- 享受表演过程

#### 第2天：站姿弹唱
**理论**（30分钟）：
- 站立持琴姿势
- 吉他背带的使用
- 站姿与坐姿的区别

**练习**（60分钟）：
- 调整背带长度
- 站立弹奏练习
- 保持稳定性
- 熟悉的歌曲站立弹唱

#### 第3天：表情管理
**理论**（30分钟）：
- 表情与音乐的配合
- 自然的表情
- 避免过度表演

**练习**（60分钟）：
- 对着镜子练习
- 录像自我检查
- 表情与音乐配合

#### 第4-7天：《平凡之路》
**歌曲**：平凡之路 - 朴树
**难度**：★★★☆☆

**和弦进行**：
```
C - Em - Am - Em
F - G - C - G
```

**练习内容**：
- 和弦配合
- 情感表达（重点）
- 舞台表现
- 站立弹唱
- 完整演绎

**表演要点**：
- 这是一首情感深沉的歌
- 需要投入真实情感
- 表情要自然
- 声音要有感染力

## 第15周：曲目巩固与提升

### 本周目标
- [ ] 巩固所有曲目
- [ ] 学会基础扒带
- [ ] 完成《斑马斑马》

### 每日学习计划

#### 第1天：如何自学新歌
**理论**（30分钟）：
- 听歌分析和弦
- 找谱子的方法
- 简化复杂和弦
- 选择合适的调

**步骤**：
1. 多听原唱，熟悉旋律
2. 找到歌曲的调式
3. 分析和弦进行
4. 选择节奏型
5. 分段练习
6. 完整演绎

#### 第2天：扒带技巧入门
**理论**（30分钟）：
- 什么是扒带
- 基础扒带方法
- 听辨和弦技巧

**练习**（60分钟）：
- 听简单歌曲
- 尝试听出和弦
- 对照谱子验证

#### 第3天：编配简单伴奏
**理论**（30分钟）：
- 和弦编配原则
- 常用和弦进行
- 节奏型选择

**练习**（60分钟）：
- 为简单旋律配和弦
- 选择合适的节奏型
- 创作简单伴奏

#### 第4天：曲目复习

- 复习所有学过的歌曲
- 每首歌完整弹唱一遍

- 重点练习薄弱环节

#### 第5-7天：《斑马斑马》

## 第16周：汇报演出准备

### 本周目标
- [ ] 准备3-5首拿手曲目
- [ ] 完成一次完整演出
- [ ] 录制演出视频

### 每日学习计划

#### 第1天：演出前准备
**理论**（30分钟）：
- 曲目选择
- 演出顺序安排
- 心理准备
- 应急预案

**曲目选择原则**：
- 选择最熟练的歌曲
- 快慢歌搭配
- 考虑情感起伏
- 3-5首为宜

**练习**（60分钟）：
- 确定演出曲目
- 安排演出顺序
- 串联练习

#### 第2天：设备调试
**理论**（30分钟）：
- 吉他检查
- 调音
- 变调夹准备
- 拨片准备

**练习**（60分钟）：
- 完整设备检查
- 调音练习
- 快速换弦练习（如需要）

#### 第3天：心理调节
**理论**（30分钟）：
- 克服紧张
- 建立自信
- 享受表演
- 失误应对

**练习**（60分钟）：
- 模拟演出场景
- 心理建设
- 放松训练

#### 第4-5天：完整曲目串烧
**练习**（每天90分钟）：
- 按演出顺序完整演奏
- 不要中断
- 模拟真实演出
- 计时练习

**演出曲目建议**：
1. 开场：《童年》（轻松欢快）
2. 第二首：《成都》（民谣风格）
3. 第三首：《晴天》（流行经典）
4. 第四首：《平凡之路》（情感深沉）
5. 结束：《董小姐》（民谣收尾）

#### 第6天：模拟演出
**练习**（90分钟）：
- 邀请家人朋友观看
- 完整演出
- 获取反馈
- 改进不足

**注意事项**：
- 穿着得体
- 保持微笑
- 眼神交流
- 享受过程

#### 第7天：录制视频与总结
**上午**（60分钟）：
- 录制演出视频
- 完整演绎3-5首歌曲
- 多角度拍摄

**下午**（60分钟）：
- 观看视频
- 自我评估
- 记录进步
- 规划未来
