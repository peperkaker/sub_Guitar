## 第1天：认识C大调音阶

### 理论知识（30分钟）

#### 什么是音阶？
音阶是按照音高顺序排列的一组音。C大调音阶是最基础的音阶。

#### C大调音阶的构成
**七个音**：C - D - E - F - G - A - B - C
**唱名**：Do - Re - Mi - Fa - Sol - La - Si - Do

#### 全音和半音
- **全音**：相隔两个半音（两品）
- **半音**：相隔一品

**C大调音阶的音程关系**：
```
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
```

#### C大调音阶在吉他上的位置

**第一把位（1-4品）**：
```
1弦：空弦(E) - 1品(F) - 3品(G)
2弦：空弦(B) - 1品(C) - 3品(D)
3弦：空弦(G) - 2品(A)
4弦：空弦(D) - 2品(E) - 3品(F)
5弦：空弦(A) - 2品(B) - 3品(C)
6弦：空弦(E) - 1品(F) - 3品(G)
```

### 实践练习（60分钟）

#### 练习1：单弦音阶（25分钟）

**3弦上的C大调音阶**：
```
3弦：空弦(G) - 2品(A) - 4品(B) - 5品(C) - 7品(D) - 9品(E) - 10品(F) - 12品(G)
```

**练习方法**：
1. 从空弦开始，逐个音弹奏
2. 每个音弹2次
3. 上行：G-A-B-C-D-E-F-G
4. 下行：G-F-E-D-C-B-A-G
5. 重复10遍

**2弦上的C大调音阶**：
```
2弦：空弦(B) - 1品(C) - 3品(D) - 5品(E) - 6品(F) - 8品(G) - 10品(A) - 12品(B)
```
- 同样练习方法

#### 练习2：跨弦音阶（20分钟）

**从5弦到1弦的C大调音阶**：
```
5弦3品(C) → 4弦空弦(D) → 4弦2品(E) → 4弦3品(F) →
3弦空弦(G) → 3弦2品(A) → 2弦空弦(B) → 2弦1品(C)
```

**练习步骤**：
1. 慢速弹奏，确保每个音清晰
2. 注意换弦的流畅性
3. 上行下行各练习5遍

#### 练习3：节拍器配合（15分钟）
- 60 BPM，每拍一个音
- 完整音阶上行下行
- 重复5遍

### 今日总结
- [ ] 理解C大调音阶构成
- [ ] 完成单弦音阶练习
- [ ] 完成跨弦音阶练习

## 第2天：六线谱入门

### 理论知识（30分钟）

#### 什么是六线谱？
六线谱是专门为吉他设计的记谱法，六条线代表吉他的六根弦。

#### 六线谱的读法
```
E |----------0---1---3---|  (1弦 - 最细)
B |------0---1---3-------|  (2弦)
G |--0---2---4-----------|  (3弦)
D |----------------------|  (4弦)
A |----------------------|  (5弦)
E |----------------------|  (6弦 - 最粗)
```

**数字含义**：
- **0**：空弦（不按品，直接拨弦）
- **1**：按1品
- **2**：按2品
- **3**：按3品
- 以此类推

#### 六线谱的节奏标记
```
E |--0-------0-------0-------0-------|
B |----------------------------------|
   1   2   3   4   1   2   3   4
  (四分音符，每拍一个音)
```

#### 常见符号
- **×**：切音/闷音
- **h**：击弦（hammer-on）
- **p**：勾弦（pull-off）
- **↓**：下拨
- **↑**：上拨

## 第4天：《欢乐颂》旋律

### 理论知识（30分钟）

#### 《欢乐颂》简介
- 贝多芬第九交响曲主题
- 旋律优美，节奏明快
- 音域适中，适合练习

#### 《欢乐颂》旋律分析
**第一句**：E E F G G F E D C C D E E D D
**节奏**：4/4拍

**音符位置**：
- C：2弦1品
- D：2弦3品
- E：4弦2品或2弦5品
- F：4弦3品或1弦1品
- G：3弦空弦

G |--0---0---------------------------|
D |--2---2---3-------------------3---|
A |----------------------------------|
E |----------------------------------|
   E   E   F   G   G   F   E
```

**练习方法**：
- 逐个音弹奏
- 确认位置
- 重复5遍

#### 练习2：完整第一句（25分钟）

**完整谱**：

## 第5天：节奏训练

### 理论知识（30分钟）

#### 音符时值
**全音符**：4拍
**二分音符**：2拍
**四分音符**：1拍
**八分音符**：半拍

#### 节奏型
**四分音符节奏**：
```
拍子：1   2   3   4
音符：♩   ♩   ♩   ♩
```

**八分音符节奏**：
```
拍子：1  &  2  &  3  &  4  &
音符：♪ ♪ ♪ ♪ ♪ ♪ ♪ ♪
```

#### 休止符
- **四分休止符**：停1拍
- **八分休止符**：停半拍

### 实践练习（60分钟）

#### 练习1：四分音符练习（20分钟）

**C大调音阶 - 四分音符**：
```
60 BPM，每个音1拍
C - D - E - F - G - A - B - C
重复10遍
```

#### 练习2：八分音符练习（20分钟）

**C大调音阶 - 八分音符**：
```
60 BPM，每个音半拍
C C D D E E F F G G A A B B C C
重复5遍
```

#### 练习3：混合节奏（20分钟）

**节奏模式1**：
```
C(1拍) - D(1拍) - E E(各半拍) - F(1拍)
重复10遍

**节奏模式2**：
```
C C(各半拍) - D(1拍) - E(1拍) - F F(各半拍)
重复10遍
```
